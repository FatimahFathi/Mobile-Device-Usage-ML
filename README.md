## Machine Learning Problem
This project analyzes mobile device usage data to understand and predict user behavior patterns using machine learning techniques.

## Dataset Description
The dataset contains information about mobile device usage, including age, gender, screen time, and app usage statistics.  
It includes more than 10,000 records and multiple features related to user behavior.

Dataset source:
https://github.com/Mikhthad/Mobile-Device-Usage-and-User-Behavior-Dataset
